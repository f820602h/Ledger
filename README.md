# Vue Cli + Express 4 + firebase + HighCharts 記帳本

## 「線上記帳本」應用程式
嘗試結合 Vue Cli3 及 Express 4 建構前後端分離的 SPA 應用架構，並且搭配 Google firebase 資料庫以及 HighCharts 圖表庫來作資料的儲存及圖像化。

## 技術資源
* Vue.js
* node.js
* firebase
* HighCharts
* heroku

## 本次挑戰與學習
* 架設後端及API設計
* NoSQL DB（firebase）
* 部署網站至 heroku
* session運用


## 注意
因部署於 heroku 上，將有機器休眠的情況，若開啟網頁時 Pending 時間過長為正常現象，請見諒。
